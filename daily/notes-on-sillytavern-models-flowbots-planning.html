the core of Flowbots centers around a Ruby Implementation of the Nano Bots Specification: small, AI-powered bots that can be easily shared as a single file, designed to support multiple providers and Jongleur, which is a is a process scheduler and manager. It allows its users to declare a number of executable tasks as Ruby classes, define precedence between those tasks and run each task as a separate process.

Jongleur is particularly useful for implementing workflows modelled as a DAG (Directed Acyclic Graph), but can be also used to run multiple tasks in parallel or even sequential workflows where each task needs to run as a separate OS process.

Stateful Multi Actor-Agent Workflows using Nano Bots, small, AI-powered bots that can be easily shared as a single file and Jongleur, which a number of inter-dependent tasks and runs them as separate processes, parallelising where possible.

The Jongleur "Actors" manage tasks that are handled by "Agentflows" which are chains of Agent modules using the low-code tool "FlowiseAI" -- Datasets, evals and prompt management are handled by Langfuse, an open source LLM Engineering Platform
