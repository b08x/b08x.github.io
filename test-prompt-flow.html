Usage Instructions

<div
  data-island="PromptFlowDiagram"
  data-props='{"initialStep": 0}'
></div>

Props:

  • initialStep (optional, default: 0): Starting animation step
    • 0 = Blank state
    • 1 = Basic prompt visible
    • 2 = User Profile injected
    • 3 = Memory injected
    • 4 = Web data injected
    • 5 = RAG/MCP data injected (complete)
