SynACK: To generate a log spectrogram showing notes using ffplay (a part of the FFmpeg suite), you would use the showspectrumpic filter. This filter generates a spectrogram on a logarithmic scale, which is more suitable for visualizing musical content as it better represents how humans perceive sound.

Here's an example command:

ffplay -f lavfi "amovie=input.mp3,showspectrumpic=s=1280x720:mode=separate:color=intensity:scale=log"

In this command:
