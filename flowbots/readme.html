Flowbots

Flowbots is an advanced text processing and analysis system that combines the power of nano-bots, workflow orchestration, and natural language processing to provide a flexible and powerful tool for document analysis and topic modeling.

Features

  • Text processing workflows for individual files and batch processing
  • Advanced NLP capabilities including tokenization, part-of-speech tagging, and named entity recognition
  • Topic modeling with dynamic model training and inference
  • Flexible workflow system using Jongleur for task orchestration
  • Redis-based data persistence using Ohm models
  • Custom nano-bot cartridges for specialized AI-powered tasks
  • Robust error handling and logging system
  • User-friendly CLI interface

Prerequisites

  • Ruby 2.7+
  • Redis server
  • Various gem dependencies (see Gemfile)

Installation

  1. Clone the repository:

    git clone https://github.com/yourusername/flowbots.git
cd flowbots

  2. Install dependencies:

	bundle install

  1. Set up environment variables:

    • REDIS_HOST: Your Redis server host
    • TOPIC_MODEL_PATH: Path to save/load topic models
    • SPACY_MODEL: Name of the spaCy model to use (e.g., "en_core_web_sm")

  2. Ensure Redis is running on your system.

Usage

Running Workflows

To list and run available workflows:

./exe/flowbots workflows

Training a Topic Model

To train a topic model on a folder of text files:

./exe/flowbots train_topic_model /path/to/folder

Processing a Single Text File

To process a single text file through the text processing workflow:

./exe/flowbots process_text /path/to/file.txt

Project Structure

  • bin/: Development scripts
  • exe/: Executable files
  • lib/: Main application code
    • components/: Core system components
    • processors/: Text and NLP processors
    • tasks/: Individual workflow tasks
    • workflows/: Workflow definitions
  • nano-bots/cartridges/: Nano-bot cartridge definitions
  • models/: Saved model files

Key Components

Workflows

  • TextProcessingWorkflow: Processes individual text files
  • TopicModelTrainerWorkflow: Batch processes multiple files for topic modeling

Processors

  • TextSegmentProcessor: Segments text into meaningful units
  • TextTokenizeProcessor: Tokenizes text segments
  • NLPProcessor: Performs NLP tasks like POS tagging and NER
  • TopicModelProcessor: Handles topic modeling operations

Tasks

Various tasks for file loading, text preprocessing, NLP analysis, filtering, and result display.

Extending Flowbots

Adding New Workflows

Create a new workflow file in lib/workflows/ and define your workflow logic. Remember to update the Workflows class to include your new workflow.

Creating Custom Nano-bot Cartridges

Add new cartridge YAML files in nano-bots/cartridges/@yourusername/cartridges/ to define specialized AI agents for your workflows.

Implementing New Tasks

Create new task classes in lib/tasks/ that inherit from Jongleur::WorkerTask to add new functionality to your workflows.
