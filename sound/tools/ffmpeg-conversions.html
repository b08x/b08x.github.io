ffmpeg conversions

To extract audio from MKV or MP4 video files into a WAV format compatible with SoX using FFmpeg, you can follow these steps. The WAV format is typically uncompressed, which ensures compatibility with SoX and preserves the highest possible quality.

Basic Command to Extract Audio

To extract audio from a video file and save it as a WAV file, you can use the following FFmpeg command:

ffmpeg -i input_video.mkv -acodec pcm_s16le -ac 2 output_audio.wav

Explanation of the Command

  • -i input_video.mkv: Specifies the input video file.
  • -acodec pcm_s16le: Sets the audio codec to PCM signed 16-bit little-endian, which is a common uncompressed audio format.
  • -ac 2: Sets the number of audio channels to 2 (stereo).
  • output_audio.wav: Specifies the output file name and format.

Handling Multiple Audio Streams

If your video file contains multiple audio streams, you can specify which stream to extract using the -map option. First, identify the streams in your video file:

ffmpeg -i input_video.mkv

This command will list all the streams in the file. For example, if the desired audio stream is Stream #0:1, you can extract it using:

ffmpeg -i input_video.mkv -map 0:1 -acodec pcm_s16le -ac 2 output_audio.wav

Ensuring Compatibility with SoX

The above command should produce a WAV file that is compatible with SoX. SoX typically works well with standard PCM WAV files, which is what the pcm_s16le codec produces.

filename="${fbasename%.*}"

local outfile="${filename}.mp4"
