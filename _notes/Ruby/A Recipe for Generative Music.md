## a recipe for generative music:


1. choose chords.
2. create matrix of notes from chords and rearrange rows in each column to minimize distance between columns.
3. render each row, one note at a time, with some instrument/synthesizer into a looper that stores each row as its own track.
4. add articulation to each loop track by modulating its volume with a simple lfo.

<ul class="chat-bubble">
  <li>
    <p><strong>Step 1: Choose Chords</strong></p>
    <p>Select the chords that will form the basis of your music. These chords will determine the overall harmony and tonality of your composition.</p>
    <p><strong>Step 2: Create a Matrix of Notes from Chords</strong></p>
    <p>Arrange the notes from the chosen chords into a matrix. Each row of the matrix will represent a different chord, and each column will represent a different note.</p>
    <p><strong>Step 3: Rearrange Rows to Minimize Distance</strong></p>
    <p>Rearrange the rows of the matrix in each column to minimize the distance between the columns. This will create a more cohesive and melodic sound.</p>
    <p><strong>Step 4: Render Each Row with an Instrument</strong></p>
    <p>Use an instrument or synthesizer to render each row of the matrix, one note at a time. Record each row as a separate track in a looper.</p>
    <strong>Step 5: Add Articulation with an LFO</strong>
    <p>Add articulation to each loop track by modulating its volume with a simple LFO (low-frequency oscillator). This will create a more dynamic and interesting sound.</p>
