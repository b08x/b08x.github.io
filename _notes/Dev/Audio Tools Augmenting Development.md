**Audio Tools Augmenting Development**

- **Code Sonification:** Use audio production tools to turn data from development processes into sound. This could serve several purposes:
- **Auditory monitoring:** Represent the state of a running program or build process as soundscapes, providing another way to perceive system activity
- **Debugging Aid:** Translate error logs or code structures into sonic patterns to help identify potential issues in a novel way.
