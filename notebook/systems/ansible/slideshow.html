# Ansible Introduction and Expertise Levels

## What is Ansible?

- Open-source automation tool
- Simplifies:
  - Configuration management
  - Application deployment
  - Task automation
  - Orchestration
- Uses simple, human-readable YAML language

slideshow

Why Use Ansible?

  • Simplicity: Uses SSH, no agents required
  • Agentless: No software/daemons on managed nodes
  • Idempotent: Applying same config multiple times doesn't change result
  • Declarative: Describe desired state, Ansible figures out how to achieve it
