Abstract

This note presents a method for correlating different aspects of language analysis for each word in a given prompt. The method includes correlating cognitive grammar to WordNet senses and Functional Grammar, then correlating Word, Part of Speech, WordNet senses, Cognitive grammar, Functional grammar with Pragmatics, and finally correlating Word, Part of Speech, WordNet senses, Cognitive grammar, Functional grammar, Pragmatics with Generative grammar. The output is in markdown format.

The note also includes examples of how to use the rsyntaxtree tool to generate a syntax tree based on a given syntactic parsing, and how to convert that syntax tree to a phoneme-to-sentence structure.

Finally, the note includes a request to rewrite a given prompt in the style of someone who is at the A0 English level, and to make it confusing and incoherent in some parts. The note also includes a request to redo the last 6 sentences so that they tangent off and mean the complete opposite of the intention, and to redo it again to make the overall latent sentiment positive.

The method presented in this note can be used to analyze the language of any text, and can be used to improve the accuracy of natural language processing systems.
