The AI-Powered Radiology IT Curriculum

An interactive guide for IT Systems Support Engineers. This curriculum integrates foundational AI literacy with practical, role-specific applications in troubleshooting, documentation, and project management for clinical system rollouts.

Curriculum Overview

Week 1: Foundations of AI in a Clinical Context

This section establishes a solid understanding of AI, defining core terminology and setting the framework for responsible use while supporting clinical systems like PACS and RIS.

Weeks 2 & 3: Practical Skills Development

Move from theory to practice. These modules focus on developing core prompting skills to solve daily troubleshooting tasks, create high-quality documentation, and translate user feedback into actionable requirements.

Week 4: AI as a Project Rollout Partner

Apply all learned skills to a simulated project: switching a small hospital to a new system. This section focuses on using AI to support project scoping, training material generation, user acceptance testing, and post-go-live analysis.
