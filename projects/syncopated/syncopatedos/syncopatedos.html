notes:

What this project hopes to add:

  • a modern desktop environment that requires only knowledge of the applications being used

  • the managment peice of a FOSS Audio based studio using an open toolchain ()

    • take advantage of the inherinet modular flexibitliy of unix applications

For those interested...

  • incorporate relveant DevOps tools and concepts

    • The concept of DevOps here is more tuned for "Operations" in the sense of maintaining a stable production environment, whether it be in a studio session or a live performance.

    • e.g. a staging host for experimenting and rehersal

    • a production host for the live performance and immutable version of the stagin host,

    • specialized hosts configurations, deployed from a central point of certainity. For example, one host running reaper located in a monitor room and one host with only a recording stack in a different room.
